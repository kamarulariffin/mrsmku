# Smart360 PWA Icons

Letakkan ikon berikut untuk "Add to Home Screen" (Android & iOS):

- **icon-192x192.png** – 192×192 px, PNG
- **icon-512x512.png** – 512×512 px, PNG

Gunakan identiti jenama Smart360 (logo, warna theme #0f766e). Ikon mestilah **maskable** (padding selamat untuk mask icon).

Jika fail ini tiada, manifest akan tetap berfungsi dengan fallback `favicon.ico`, tetapi A2HS mungkin tidak memaparkan ikon penuh.

**Nota:** Ikon default (192×192 dan 512×512) disertakan. Nama aplikasi, warna tema dan URL ikon boleh diubah dalam **Tetapan Sistem → Smart360** (disimpan dalam pangkalan data).
